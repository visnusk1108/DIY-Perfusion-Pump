# DIY Perfusion Pump

## Overview
A low-cost automated perfusion pump for controlled fluid delivery in biomedical applications. The system uses a motor-driven lead screw to convert rotational motion into linear actuation for precise flow control.

## Key Features
- Lead screw–based linear actuation  
- Motor-driven flow control mechanism  
- Modular, low-cost design for prototyping  
- Designed with DFM/DFA principles  

## Technologies
Fusion 360 · Arduino · Embedded Systems · Basic Data Analysis  

## Project Structure
- **CAD/** – 3D models (Fusion 360, STEP, STL)  
- **Electronics/** – Circuit diagrams and code  
- **Documentation/** – BOM, design notes, assembly guide  
- **Images/** – Renders and prototype images  

## Working Principle
A motor rotates the lead screw, generating linear motion that compresses the perfusion mechanism to regulate fluid flow.
